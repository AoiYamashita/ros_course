Docker 概念編

dockerとは

浅く説明するとdockerはコンテナといわれるアプリケーションや実行環境をひとまとめにしたものを管理するソフトウェアです
深い説明はサーバーなどのバックエンド側の知識がないとつらいところなので調べたい人は調べてみてください

dockerを使う上で知っておくべき概念

Container

先ほどの説明にあったようにアプリケーションや実行環境を一括りにしたものだという認識で今回は大丈夫です

image

containerを構成する要素をまとめたもの又は構成の一要素

Dockerfile

image 内の要素について記述したdocker専用ファイル

build

Dockerfileからimageを作ること

imageID/containerID

dockerのimage,containerに個別に付けられるID

Dockerのよく使うコマンド

docker image list

buildされたimageのlistを表示する

docker ps

起動しているcontainerの一覧を見ることができる
-a オプションを付けると停止している物も見れる

docker build directory名

directory内にあるDockerfileのbuild
-t オプションを付けることでimageの名前を決められる

docker start/stop container名/containerID

containerの起動、停止

docker exec container名/containerID directory

起動中のcontainerにアクセスする
