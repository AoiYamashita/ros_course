Git Github

分散型のバージョン管理システム
チーム開発や複数のバージョン管理に便利
WSLなどのCLI上での開発には必須といえるぐらいの効力を発揮

レポジトリの作成

Gitではレポジトリを作成し、そのディレクトリ上で作業を行うことでバージョン管理や共有、共同作業を簡単に行うことができます。
