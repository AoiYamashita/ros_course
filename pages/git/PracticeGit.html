Git 実践編

ローカルレポジトリでの編集

ローカルレポジトリ上で何かしらの編集を行います
まずは編集を行うためのテキストファイルを作成します。

cd レポジトリ名
touch practice.txt

今回は「practice.txt」というテキストファイルを作成します
さらに作成したファイルに何かしらの編集を行うためコードエディタで編集を行います
今回はcode spaceでの編集を前提としているのでvscodeで編集を行います

code practice.txt

何かしら編集を行ったら保存しましょう

リモートレポジトリの確認

github上で現在cloneしているレポジトリを見てみましょう
まだ、なんの変化も起こっていないはずです

リモートレポジトリに変更の適応

変更の記録

git add .

ローカルリポジトリに作業を登録

git commit -m "コミットメッセージ"

ローカルリポジトリをリモートリポジトリにアップロード

git push

リモートレポジトリの確認

これまでの作業を正しく行うことができればpractice.txtが追加されているはずです
