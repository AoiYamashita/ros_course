ubuntu 基本操作編

ubuntu上では基本的にwindowsやmacのようなGUI操作を行うことはないそのためrosなどの開発を行うために基本的な操作について学んでおく必要がある
慣れの部分が大きいのでコマンドを実行しubuntuを使っているうえで覚えることを推奨
調べれば出てくるので簡易的な紹介、今回の紹介で分からなければgoogleに頼りましょう

pwd

現在作業を行っているディレクトリの表示

cd ディレクトリ名

ディレクトリの移動を行う

ls

ディレクトリ内のファイル、ディレクトリの確認を行える

mkdir ディレクトリ名

ディレクトリを作成できる

echo "文字列"

文字列を表示できる
単体で使うことはほぼない、組み合わせたり、shell scriptに記述したりして使う

touch ファイル名.拡張子

空のファイルを作れる

cat

ファイル内容の表示

rm ファイル名

ファイル、ディレクトリの削除
むやみに使ってubuntuを壊さないように気を付けましょう

cp ファイル名 コピー先ディレクトリ

ファイルのコピー
ディレクトリのコピーなどはgitでできるようになってほしい

mv ファイル名 移動先

ファイルの移動
移動先を工夫するとファイル名の変更が行える

chmod パーミッション ファイル名

パーミッションの付与、剥奪

apt

アプリケーションのインストール、アンインストール、検索を行える

man コマンド名

マニュアルの閲覧

sudo コマンド名

管理者権限による実行、パスワードを求められることがある
